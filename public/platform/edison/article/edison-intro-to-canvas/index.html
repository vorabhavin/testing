Introduction to Canvas Web Builder

Introduction to Canvas Web Builder

by Tatiana Ugriumova, Product Owner

Building a website with a handful of clicks: introducing Canvas Web Builder

Published: Dec 9, 2021

What is Canvas Web Builder?

Canvas Web Builder is a self-service tool allowing marketers to build and edit their own web pages using pre-built components. It is ideal for brand, product and campaign sites that do not require complex content management workflows.

​​​​​​​


Why is Canvas Web Builder important to Pfizer?

Self-service and low-code solutions like Canvas Web Builder allow us to publish content more quickly - developers are not needed for publishing content, which means marketers can focus more on their message and less on the technology.

Canvas Web Builder gives us the capability to rival commercial offerings like Wix or Squarespace for ease of use. These benefits cascade all the way to the end user - patients, HCPs and the public - who use digital Pfizer products.

How does Canvas Web Builder fit into the bigger picture of Pfizer Digital?

Canvas Web Builder integrates closely with the Edison web publishing platform. When you complete your page builds in CWB, you can publish them to an Edison Lite site, meaning they are hosted on a secure and scalable in-house platform. Additionally, as Canvas integrates and works closely with Edison, accessing knowledgeable support is incredibly easy.

What problems or challenges does Canvas Web Builder solve?

  • Build time for websites: you can build a fully finished, totally compliant, correctly branded page in a couple of minutes with just a few clicks

  • Canvas speeds up the publication process - it puts control in the hands of marketers and removes dependency on development cycles

  • You can also select from a library of common components for your specific site (like headers and footers) that mean you can assemble pages very quickly

What are the benefits of Canvas Web Builder?

  • Self-service

  • No-/low-code

  • Standardized design components

  • Flexible layouts

  • Quick and easy content updates

Who would use Canvas Web Builder/who is Canvas Web Builder aimed at?

CWB is useful for a marketing team that wants to take control of their own content publication process.

What’s the big shout?

Canvas Web Builder is now available for patient sites!

How can people reach you and learn more about Canvas Web Builder?
