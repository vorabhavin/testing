Stratus is a platform ecosystem that empowers development teams ​to design, build, and deploy custom applications quickly and easily. Stratus enables you to build bespoke, native cloud applications and microservices on secure, stable and compliant Pfizer-managed infrastructure. 

Pipeline

Stratus Pipeline is a collection of capabilities that manages your code from development to production. With built-in layers of automated quality and security checks, and coding and testing standards, you can confidently deploy to production at any time.  Our pipeline tools are designed to provide the foundation for continuous integration, meaning you can deploy new features and changes multiple times per day rather than working to fixed release cycles.
