Force.com is Pfizer’s platform for building Customer Relationship Management (CRM) and complex business process applications in a “low code” configurable Platform-as-a-Service (PaaS) license model

Pipeline

Force pipeline is a suite of tools that take your applications from development to production. Run automated tests and code quality checks whenever code is committed to ensure reliable applications are deployed.  You can maintain your applications from a unified dashboard to give you control.

Accelerate

Force Accelerate provides your development team with the tools and best practice processes they need to build better applications in Force.com. 
The core components of Force.com are enhanced with specialist Pfizer-specific components that can be used in a no-code environment.

Connect

Force Connect allows you to expand out of the Force.com ecosystem, building hybrid apps using other platforms such as Stratus, Edison and Newton enabling you to provide unique user experiences. 

We also provide prebuilt data warehouses with high value datasets for Customer and Commercial Pfizer applications build on Force.com.
