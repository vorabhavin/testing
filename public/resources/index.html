Helix Modal Header

Follow the instructions below to create a trigger for your modal:

  • Add an action identifier to the modal (via the trait menu).
  • Drop in a helix-anchor/helix-button element.
  • Enter an action identifier which should be the same as the action identifier you supplied for this modal.
  • Enter an action method which should be "show".
  • Enter preview mode and ensure when the helix-anchor/helix-button are clicked you are presented with the modal.
